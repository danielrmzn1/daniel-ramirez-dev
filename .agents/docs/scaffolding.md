# Scaffolding Guide

This document provides a reference for the initial pages and components to be scaffolded for the project.

## Initial Pages to Scaffold

Create the following pages in `src/pages/`:

- `/` (Home): Should include a hero section, a brief introduction, and a list of featured projects.
- `/about`: A page describing the individual or company.
- `/projects`: A gallery or list of all projects.
- `/contact`: A page with contact information or a contact form.

## Initial Components

Create the following initial components.

### Astro Components (`src/components/astro/`)

- **Navbar:** The main site navigation.
- **Footer:** The site footer.
- **Section:** A reusable wrapper component for content sections.
- **ProjectCard:** A static card to display project information.
- **Hero:** The hero section for the home page.

### React Components (`src/components/react/`)

- **ThemeToggle:** An interactive button to toggle between light and dark themes.
- **ProjectFilter:** Interactive controls to filter or sort the projects on the `/projects` page.
